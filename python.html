Resumo de Python

Python é uma linguagem de programação muito utilizada por sua simplicidade e clareza. Excelente para iniciantes e igualmente poderosa para profissionais.

É usada em áreas como:
– Inteligência Artificial
– Automação
– Desenvolvimento Web
– Ciência de Dados
– Jogos

Sua grande força está em suas bibliotecas: NumPy, Pandas, Django, Flask, TensorFlow e muitas outras.
